# RN-LiveChat
Live Chat app using React Native && Firebase

## Installation
```
git clone https://github.com/Kochua/RN-LiveChat.git

cd RN-LiveChat/

npm install

cd ios/ && pod install

npx react-native run-ios
or
npx react-native run-android
```
